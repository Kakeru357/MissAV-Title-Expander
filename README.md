# MissAV Title Expander

MissAVの動画一覧ページなどで、省略（...）されてしまう長いタイトルをすべて表示するためのGoogle Chrome拡張機能です。

## 概要
MissAVの標準レイアウトでは、長いタイトルが2行に制限され省略表示されます。この拡張機能を導入することで、タイトルの省略を解除し、全ての文字を表示させることができます。

## 主な機能
- **タイトル全文表示**: `line-clamp` による2行制限を解除します。
- **レイアウト最適化**: タイトルが長くなってもカードの高さが自動調整され、視認性を維持します。
- **軽量動作**: JavaScriptを使用せず、CSSのみでスタイルを上書きするため、動作が非常に軽快です。

## インストール方法（開発者モード）
現在、本拡張機能はGitHubからの手動インストールに対応しています。

1. 本リポジトリをクローンするか、ZIP形式でダウンロードして解凍します。
2. Google Chromeを開き、アドレスバーに `chrome://extensions/` を入力します。
3. 右上の「**デベロッパー モード**」をオンにします。
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリックします。
5. 解凍したフォルダを選択するとインストールが完了します。

## 使い方
インストール後、[MissAV](https://missav.ai/) にアクセスするだけで自動的に適用されます。特別な設定は不要です。

## 技術仕様
- **Manifest V3** 対応
- **対象サイト**: `https://missav.ai/*`
- **適用スタイル**: CSS `Content Scripts` によるプロパティの上書き
